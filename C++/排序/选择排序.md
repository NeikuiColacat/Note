# 选择排序

**思路**   每次固定一个位置i，然后遍历i ~ n , 进行swap ， 以此类推


```cpp
#include <stdio.h>
int main(void){
    int n;
    scanf("%d",&n);
    int arr[1000];
    for(int i =0;i<n;i++)scanf("%d",&arr[i]);

    for(int i =0;i<n;i++){
        //固定好位置i，然后排序
        for(int j=i+1;j<n;j++){
            int temp=0;
            if(arr[i] > arr[j]){
                temp = arr[i];
                arr[i] = arr[j];
                arr[j] = temp;
            }

        }
    }


}

```
