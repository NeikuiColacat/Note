# 模运算

## `A%B=A-A/B*B`
在c++中:
- 正数 mod 负数 为正数
- 负数 mod 负数 为负数
- 负数 mod 正数 为负数

- (a + b) % p = (a % p + b % p) % p 
- (a - b) % p = (a % p - b % p ) % p 
- (a * b) % p = (a % p * b % p) % p 
- (a^b) % p = ((a % p)^b) % p 
结合律：
- ((a+b) % p + c) % p = (a + (b+c) % p) % p 
- ((a*b) % p * c)% p = (a *(b*c)%p) % p 
交换律：
- (a + b) % p = (b+a) % p 
- (a * b) % p = (b * a) % p 
分配律：
- ((a +b)% p * c) % p = ((a * c) % p + (b * c) % p) % p 
重要定理：
若a≡b (% p)，则对于任意的c，都有(a + c) ≡ (b + c) (%p)；
若a≡b (% p)，则对于任意的正整数c，都有(a * c) ≡ (b * c) (%p)；
若a≡b (% p)，c≡d (% p)，则 (a + c) ≡ (b + d) (%p)，(a - c) ≡ (b - d) (%p)，
(a * c) ≡ (b * d) (%p)；
