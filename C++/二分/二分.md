内置函数:
升序：
upper_bound(arr.begin(),arr.end(),a)查找数组第一个大于a的元素，返回地址
lower_bound(arr.begin(),arr.end(),a) 查找数组中第一个大于或等于a的元素，返回地址
降序反之
变成小于
小于等于

-------------
```c++

while (left <= right) {
	if (x > arr[mid]) {
		left = mid + 1;
		mid = (left + right) / 2;
	}
	else if (x < arr[mid]) {
		right = mid - 1;
		mid = (left + right) / 2;
	}
	else if (x == arr[mid])break;
}
```
