## 状态转移方程 二维
f[i][j] 表示从a字符串中前i个字符，b字符串中前j个字符的最长公共子序列长度
```cpp
#include<bits/stdc++.h>
using namespace std;
const int N =1010;
char a[N], b[N];
int f[N][N];
signed main() {
	int n, m;
	cin >> n >> m >> a + 1 >> b + 1;
	for (int i = 1; i <= n; i++) {
		for (int j = 1; j <= m; j++) {
			f[i][j] = max(f[i - 1][j], f[i][j - 1]);
			if (a[i] == b[j])f[i][j] = max(f[i][j], f[i - 1][j - 1] + 1);
		}
	}
	cout << f[n][m];
}
```
