动态内存分配
malloc malloc以字节为单位
free
俩函数
在stdlib头文件里
数组变量是常量指针
int number = 3;
int\* p = &number;
此时 p\[0\] == number

指针 <sub>+ 1 加上的并不是1，而是指针变量的sizeof,由指针变量类型的大小单元格决定</sub>
int 4 个字节
int a = 1;
int \*p = &a;
printf ("%d",p + 1);
输出结果比p 大 4

指针相减 等于 相减后的差除以sizeof
即相差几个存储单元

传进函数的数组其实是一个指针
