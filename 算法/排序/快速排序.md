# 不稳定  平均为 $O(nlogn)$ 最坏为$O(n^2)$
```cpp
void quick_sort(int* a,int l,int r){
    if(l >= r)return;
    int i = l - 1;
    int j = r + 1;
    int x = a[l];//x不能取a[r],会有边界问题，比如数组为1，2
    while(l<r){
        do i++;while(a[i] < x);//注意不能取等于，保证左边右边的指针不会越过标志数
        do j--;while(a[j] > x);
        if(l<r) swap(a[i],a[j]);
    }
    quick_sort(a,l,j);
    quick_sort(a,j+1,r);
}
```
